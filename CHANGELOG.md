## [1.0.0] — 2026-05-06

### Added

- Floating task panel that lives in the menu bar, always one click away
- Focus mode — isolate a single task and zero in until it's done
- Keyboard-first input — press `Cmd-N` to start typing, `Esc` to dismiss
- Progress tracking — completion rate at a glance with an animated progress line
- Drag-to-reorder tasks
- Inline editing with double-click
- Copy task title to clipboard
- Context menu with focus, edit, copy, and delete actions
- Panel position persists across launches
- Focus state persists across app restarts
- Animated transitions for completion, focus, and panel interactions
- Lightweight — no accounts, no sync, no subscriptions
